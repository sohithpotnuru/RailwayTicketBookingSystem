How to Run in VS Code:
1. Install JDK 8 or above.
2. Open this folder in VS Code.
3. Install the "Extension Pack for Java".
4. Open the src folder.
5. Open Main.java.
6. Click the Run button at the top right, or run these commands in terminal:
   javac src/*.java
   java -cp src Main

Project Files:
- Main.java
- User.java
- Admin.java
- Passenger.java
- Train.java
- Ticket.java
- PaymentService.java
- DataManager.java
- BookingSystem.java

Data Files:
When you run the program, it creates:
- users.dat
- trains.dat
- tickets.dat
These save data between runs.
